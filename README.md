📊 Sales Performance Dashboard

## Overview

This project is an interactive Power BI dashboard designed to analyze sales performance across products, categories, and regions.

## Features

- 💰 Total Sales KPI
- 📦 Total Orders KPI
- 📈 Total Profit KPI
- 📊 Product-wise Sales
- 🍩 Category-wise Sales
- 🌍 Region-wise Sales
- 📅 Monthly Sales Trend
- 🔍 Interactive slicers for Category, Month, and Region

## Tools Used

- Power BI Desktop
- Microsoft Excel
- DAX
